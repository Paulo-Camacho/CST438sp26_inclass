# Video Game Recommendation App

A web application that recommends video games to users based on selected preferences such as genre, platform, and play style. Users can explore games, view details, and receive random recommendations using a public game API.

## Project Overview

This application helps users discover new video games by applying filters and recommendations.  
The app is built as part of an in-class group project and follows GitHub issue tracking and pull request workflows.

Initial planning, features, and issue breakdown are documented in the project proposal.

---

## API Used

- **FreeToGame API**  
  https://www.freetogame.com/api-doc

The API provides:
- Game URL  
- Short description  
- Genre  
- Platform  
- Developer  
- Publisher  
- Release date

## Planned Features

Based on our project proposal, the following features were identified:

1. User sign-on page (login)
   - Create account
   - Forgot password
2. Admin login
   - Add or delete users
3. Welcome / home page
4. History of past searches
5. Filter games by genre
6. Personal game reviews
7. Link to game profile URL
8. Random game suggestion
9. Additional feature TBD
